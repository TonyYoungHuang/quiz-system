# 小程序提审说明（2026-04-08）

## 本次调整概述

本次主要完成以下几类优化与修复：

1. 修正 CTCSOL 历年真题的归属方式，将原先误放在“专题训练”中的 `2025年6月真题` 与 `2025年9月真题` 迁移到“模拟考试 / 试卷”路径。
2. 优化“模拟考试”页面 UI，使其与“专题训练”页面在卡片结构、按钮样式、标签区样式上保持一致，统一小程序内专题、模拟等训练入口的视觉语言。
3. 增强题库导入能力，支持知识点型 `SHORT` 题在专题训练中作为“参考题 / 知识点题”导入，并通过 `autoFullScore` 实现无需作答即可默认得分。
4. 新增《中国文化要略复习笔记》本地 PDF 转题库工具，可生成适配后台 JSON 导入格式的专题题库数据。

## 可直接用于提审的简要说明

本次版本主要优化了刷题小程序中“模拟考试”与“专题训练”的页面一致性，统一了题目入口的页面样式与交互展示；同时修正了 CTCSOL 历年真题的归属路径，将 2025 年 6 月真题、2025 年 9 月真题调整到“模拟考试”模块展示。另对题库导入链路做了兼容增强，支持知识点型题目按参考题方式导入与展示，不影响现有正常做题流程。

## 影响范围

- 小程序前端：
  - `模拟考试` 页面样式统一
  - 专题 / 试卷下的 `SHORT` 知识点题默认按参考题计分
- 管理后台：
  - JSON 导入链路支持保留 `autoFullScore`
- 云函数：
  - 新增一次性专题转试卷迁移函数
- 本地开发工具：
  - 新增中国文化专题 PDF 转导入 JSON 脚本

## 用户可见变化

- “模拟考试”页面视觉风格现已与“专题训练”统一。
- CTCSOL 的 `2025年6月真题`、`2025年9月真题` 已在“模拟考试”中展示，不再占用“专题训练”题量。
- “中国文化系列 / 中国文化要略”这类知识点专题可同时包含：
  - 普通填空题
  - 知识点型参考题（无需填写，提交即得分）

## 风险与注意事项

- 历年真题迁移使用的是一次性云函数，后续新增真题时应直接绑定 `paperId`，不要再绑定 `topicId`。
- 中国文化专题导入 JSON 属于本地题库资料，当前仅本地保留，不纳入本次 GitHub 提交。
- 如需继续提升“中国文化”专题的刷题感，可后续将部分 `SHORT` 参考题再细化为单选或标准填空题。
