# 云开发版管理后台部署指南

## 项目结构

```
miniprogram/
├── cloudfunctions/          # 云函数
│   ├── adminLogin/         # 管理员登录
│   ├── adminGetStats/      # 获取统计数据
│   ├── adminGetExams/      # 获取科目列表
│   ├── adminCreateExam/    # 创建科目
│   ├── adminImportQuestions/ # 批量导入题目
│   ├── adminGetCodes/      # 获取激活码列表
│   └── adminGenerateCodes/ # 批量生成激活码
│
└── admin-web/              # 管理后台前端
    └── index.html          # 单页应用（包含所有功能）
```

---

## 部署步骤

### 第一步：上传管理云函数

在微信开发者工具中，依次上传以下云函数：

1. **右键点击** `cloudfunctions/adminLogin`
2. 选择**「上传并部署：云端安装依赖」**
3. 等待上传完成

重复以上步骤，上传所有管理云函数：
- ✅ `adminLogin`
- ✅ `adminGetStats`
- ✅ `adminGetExams`
- ✅ `adminCreateExam`
- ✅ `adminImportQuestions`
- ✅ `adminGetCodes`
- ✅ `adminGenerateCodes`

### 第二步：开通静态网站托管

1. 点击工具栏**「云开发」**按钮
2. 点击**「静态网站托管」**
3. 点击**「开通」**
4. 选择**「按量付费」**（免费额度足够）
5. 点击**「确定」**

### 第三步：上传管理后台

1. 在静态网站托管页面，点击**「上传文件」**
2. 选择 `admin-web/index.html` 文件
3. 上传完成后，会获得一个访问地址，例如：
   ```
   https://xxx.cloudbase.net/index.html
   ```

### 第四步：测试访问

1. 在浏览器中打开访问地址
2. 输入默认密码：`admin123`
3. 登录成功后即可管理数据

## 功能说明

### 1. 数据概览
- 查看科目总数
- 查看题目总数
- 查看激活码总数
- 查看用户激活数量

### 2. 科目管理
- 添加新科目（名称、分类、图标、排序）
- 删除科目
- 查看每个科目的题目数量

### 3. 题目管理
- 批量导入题目（CSV格式）
- 支持单选题、多选题、判断题
- 自动解析选项和答案

### 4. 激活码管理
- 批量生成激活码
- 查看激活码列表
- 筛选科目查看
- 删除未使用的激活码

## CSV导入格式

### 格式说明

```csv
题型,题目内容,A选项,B选项,C选项,D选项,答案,解析
单选题,以下哪个是正确的？,选项A的内容,选项B的内容,选项C的内容,选项D的内容,A,这是解析内容
多选题,以下哪些是正确的？,选项A的内容,选项B的内容,选项C的内容,选项D的内容,AB,这是解析内容
判断题,地球是圆的吗？,,,,,A,这是解析内容
```

### 字段说明

| 字段 | 说明 | 示例 |
|------|------|------|
| 题型 | 单选题/多选题/判断题 | 单选题 |
| 题目内容 | 题目描述 | 以下哪个是正确的？ |
| A选项 | 选项A的内容 | 选项A的内容 |
| B选项 | 选项B的内容 | 选项B的内容 |
| C选项 | 选项C的内容 | 选项C的内容 |
| D选项 | 选项D的内容 | 选项D的内容 |
| 答案 | A/B/C/D 或 AB/ABC（多选）或 A/B（判断） | A |
| 解析 | 答案解析（可选） | 这是解析内容 |

### 注意事项

1. 判断题的选项可以留空
2. 多选题答案可以是：AB、ABC、ABCD等
3. 判断题答案：A=正确，B=错误
4. 支持管道符(\|)分隔选项
5. 每行一道题，第一行为表头（可选）

## 常见问题

### Q1: 如何修改管理员密码？

A: 首次登录后，密码存储在云数据库的 `config` 集合中。可以在云开发控制台手动修改：
```
数据库 → config 集合 → 找到 key=admin_password 的记录 → 修改 value 字段
```

### Q2: 批量导入题目数量限制？

A: 云函数单次调用最多处理20条记录。如果需要导入更多题目，前端已自动分批处理，理论上可以无限导入。

### Q3: 激活码格式是什么？

A: 激活码为8位大写字母和数字组合，例如：`A3B5C7D9`。避免易混淆字符（如0/O、1/I）。

### Q4: 如何删除测试数据？

A: 在云开发控制台的数据库中，直接删除对应集合的记录即可。

### Q5: 托管域名可以自定义吗？

A: 可以。在静态网站托管中，可以绑定自定义域名（需要已备案）。
