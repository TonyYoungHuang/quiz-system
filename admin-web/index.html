导入格式说明

简单客观题建议使用 CSV；材料题、配图题、含子题结构的题库建议使用 JSON。

题型,题目内容,A选项,B选项,C选项,D选项,E选项,F选项,G选项,答案,解析
单选题,以下哪个属于学习计划执行动作？,开始复盘,删除课程,取消笔记,关闭提醒,,,,A,开始复盘属于实际执行动作
多选题,以下哪些属于高效刷题方法？,分专题练习,错题复盘,只看答案,定期模考,阶段复盘,限时练习,真题复盘,ABEFG,专题练习+错题复盘+阶段复盘+限时练习更有效
判断题,题库刷题前先明确考试科目是合理的,,,,,,,,A,先选定科目和范围更利于整理题库
填空题,企业常见的客户关系管理缩写是？,,,,,,,,CRM,常见缩写为 CRM
简答题,请简述刷题后如何整理错题,,,,,,,,先分类记录错因，再定期复盘并二次练习,写出关键步骤即可
计算题,某题库共 120 题，已完成 45 题，完成率是多少？,,,,,,,,37.5%,120 题完成 45 题即 37.5%

支持题型：单选题、多选题、判断题、填空题、简答题、开放题、论述题、计算题、材料题。

CSV 适合普通题：选择题支持 A-G 7 个选项；填空题多个答案用 | 分隔；简答题/开放题/论述题/计算题可留空选项列。

JSON 适合复杂题：可导入 stem、analysis、media、children，可承载配图题、材料题、子题结构。
