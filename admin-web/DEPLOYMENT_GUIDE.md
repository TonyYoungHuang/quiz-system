# 云开发版管理后台部署指南（修订版）

## 项目概述

这是基于微信云开发的刷题系统管理后台，提供完整的数据管理功能，无需购买服务器。

## 前置条件

- 已创建微信云开发环境，并记录环境 ID（例：`cloud1-0g8twq2fde2fa6f0`）
- 已在微信开发者工具中打开项目并绑定云环境
- 云函数目录已安装依赖（首次上传时选择"云端安装依赖"）
- 管理后台前端资源已构建并可上传（若有静态资源，需一起上传）

## 部署步骤

### 第一步：上传管理云函数

在微信开发者工具中，依次上传以下云函数：

1. **右键点击** `cloudfunctions/adminLogin` 文件夹
2. 选择**「上传并部署：云端安装依赖」**
3. 等待上传完成（云函数图标会变成☁️）

重复以上步骤，上传所有 **10 个**管理云函数：

- ✅ `adminLogin` - 管理员登录
- ✅ `adminGetStats` - 获取统计数据
- ✅ `adminGetExams` - 获取科目列表
- ✅ `adminCreateExam` - 创建科目
- ✅ `adminUpdateExam` - 更新科目
- ✅ `adminDeleteExam` - 删除科目
- ✅ `adminGetQuestions` - 获取题目列表
- ✅ `adminImportQuestions` - 批量导入题目
- ✅ `adminGetCodes` - 获取激活码列表
- ✅ `adminGenerateCodes` - 批量生成激活码
- ✅ `adminDeleteCode` - 删除激活码

每个云函数上传大约需要 10-30 秒。

### 第二步：开通静态网站托管

1. 点击微信开发者工具工具栏上的**「云开发」**按钮
2. 点击左侧菜单的**「静态网站托管」**
3. 点击**「开通」**
4. 选择**「按量付费」**（免费额度足够初期使用）
5. 点击**「确定」**

### 第三步：上传管理后台

> **重要说明**：若管理后台包含 CSS/JS/图片等静态资源，必须上传整个 `admin-web/` 目录，而不是只上传 `index.html`。

1. 在静态网站托管页面，点击**「上传文件/文件夹」**
2. 选择 `admin-web/` 目录（或确保包含所有静态资源）
3. 上传完成后，会获得一个访问地址，例如：
   ```
   https://xxx.cloudbase.net/index.html
   ```

### 第四步：测试访问

1. 在浏览器中打开访问地址
2. 输入默认密码：`admin123`
3. 点击「登录」
4. 登录成功后即可管理数据

## 功能说明

### 1. 数据概览
- 查看科目总数
- 查看题目总数
- 查看激活码总数
- 查看用户激活数量

### 2. 科目管理
- **添加科目**：设置科目名称、分类、图标、排序
- **删除科目**：删除不需要的科目（相关题目和激活码也会被删除）
- **查看统计**：查看每个科目的题目数量

### 3. 题目管理
- **批量导入**：支持 CSV 格式批量导入题目
- **支持题型**：单选题、多选题、判断题
- **格式灵活**：支持逗号或管道符(`|`)分隔选项

### 4. 激活码管理
- **批量生成**：一次生成多个激活码
- **科目筛选**：按科目查看激活码
- **一键复制**：支持单个或批量复制激活码
- **删除管理**：删除未使用的激活码

## CSV 导入格式说明

### 标准格式

```csv
题型,题目内容,A选项,B选项,C选项,D选项,答案,解析
单选题,以下哪个是正确的？,选项A的内容,选项B的内容,选项C的内容,选项D的内容,A,这是解析内容
多选题,以下哪些是正确的？,选项A的内容,选项B的内容,选项C的内容,选项D的内容,AB,这是解析内容
判断题,地球是圆的吗？,,,,,A,这是解析内容
```

### 字段说明

| 字段 | 说明 | 示例 | 必填 |
|------|------|------|------|
| 题型 | 单选题/多选题/判断题 | 单选题 | 是 |
| 题目内容 | 题目描述 | 以下哪个是正确的？ | 是 |
| A选项 | 选项 A 的内容 | 选项 A 的内容 | 单选/多选必填 |
| B选项 | 选项 B 的内容 | 选项 B 的内容 | 单选/多选必填 |
| C选项 | 选项 C 的内容 | 选项 C 的内容 | 单选/多选必填 |
| D选项 | 选项 D 的内容 | 选项 D 的内容 | 单选/多选必填 |
| 答案 | A/B/C/D 或 AB/ABC（多选） | A | 是 |
| 解析 | 答案解析 | 这是解析内容 | 否 |

### 题型说明

**单选题**
- 答案为单个字母：A、B、C、D
- 示例：`单选题,以下哪个是正确的？,选项A,选项B,选项C,选项D,A,这是解析`

**多选题**
- 答案为多个字母组合：AB、ABC、ABCD
- 示例：`多选题,以下哪些是正确的？,选项A,选项B,选项C,选项D,AB,这是解析`
- 建议按 A→D 顺序填写答案，避免顺序不一致

**判断题**
- 选项可以留空
- 答案：A=正确，B=错误
- 示例：`判断题,地球是圆的吗？,,,,,A,这是解析`

### 管道符格式（备选）

如果使用管道符(`|`)分隔选项：

```csv
题型|题目内容|A选项|B选项|C选项|D选项|答案|解析
单选题|以下哪个是正确的？|选项A|选项B|选项C|选项D|A|这是解析
```

系统会自动识别两种格式。

### 注意事项

- **编码问题**：CSV 文件建议使用 UTF-8 编码保存
- **空值处理**：判断题的选项可以为空
- **批量导入**：一次可以导入成百上千道题
- **错误报告**：导入失败会显示具体失败原因
- **常见错误**：答案字段为空、答案字母超出 A-D、题型填写错误

## 使用指南

### 添加科目并导入题目

1. 登录管理后台
2. 点击左侧菜单「📚 科目管理」
3. 点击「添加科目」按钮
4. 填写科目信息：
   - 科目名称：如「对外汉语」
   - 分类：如「教师资格」
   - 图标：选择一个 emoji 图标
   - 排序：数字越小越靠前
5. 点击「确定」创建科目

### 批量导入题目

1. 点击左侧菜单「✏️ 题目管理」
2. 点击「批量导入」按钮
3. 选择刚才创建的科目
4. 准备 CSV 数据（可从 Excel 复制粘贴）
5. 粘贴到文本框中
6. 点击「开始导入」
7. 等待导入完成，查看导入结果

### 生成激活码

1. 点击左侧菜单「🔑 激活码管理」
2. 点击「生成激活码」按钮
3. 选择科目
4. 输入生成数量（1-100 个）
5. 选择来源（小红书/淘宝/拼多多/闲鱼/手动）
6. 点击「生成」
7. 生成后可以：
   - 查看生成的激活码列表
   - 点击「复制全部」一次性复制所有激活码
   - 单个激活码也可以单独复制

## 常见问题

### Q1: 如何修改管理员密码？

**A**: 首次登录后，密码存储在云数据库的 `config` 集合中。

**修改方法：**
1. 进入云开发控制台
2. 点击「数据库」
3. 找到 `config` 集合
4. 查找 `key=admin_password` 的记录
5. 修改 `value` 字段为新密码

### Q2: 批量导入题目数量有限制吗？

**A**: 云函数单次调用最多处理 20 条记录。但系统已自动分批处理，理论上可以无限导入。

**实际测试：**
- 导入 100 道题：大约 30 秒
- 导入 1000 道题：大约 3-5 分钟
- 导入 3000 道题：大约 10-15 分钟

### Q3: 激活码格式是什么？

**A**: 激活码为 8 位大写字母和数字组合，例如：`A3B5C7D9`。

**特点：**
- 避免易混淆字符（0/O、1/I）
- 大写字母
- 随机生成，不重复
- 只能使用一次

### Q4: 如何删除测试数据？

**A**: 在云开发控制台中删除即可。

**方法：**
1. 进入云开发控制台
2. 点击「数据库」
3. 选择对应集合
4. 勾选要删除的记录
5. 点击「删除」

**清空集合：**
- 可以直接在集合中删除所有记录
- 或者在集合设置中删除集合并重新创建

### Q5: 托管域名可以自定义吗？

**A**: 可以。微信云开发支持绑定自定义域名。

**前提条件：**
- 域名已完成备案
- 域名已认证

**绑定方法：**
1. 在静态网站托管中点击「设置」
2. 点击「添加域名」
3. 填写已备案的域名
4. 按提示添加 DNS 解析记录
5. 等待验证完成

### Q6: 数据安全如何保障？

**A**: 系统已实现以下安全措施：

- **登录验证**：所有管理操作需要登录 token
- **Token 过期**：2 小时自动过期，需要重新登录
- **数据隔离**：云数据库有严格的权限控制
- **操作日志**：云函数会记录操作日志

**建议：**
- 定期更换管理员密码
- 不要在公共场所访问管理后台
- 及时删除未使用的测试数据

### Q7: 如何备份数据？

**A**: 微信云开发提供自动备份功能。

**查看备份：**
1. 进入云开发控制台
2. 点击「数据库」
3. 点击「备份与恢复」
4. 可以查看备份列表和手动创建备份

**导出数据：**
- 可以通过云函数导出 JSON 格式数据
- 或使用数据库管理工具导出

### Q8: 管理后台可以多人同时使用吗？

**A**: 技术上可以，但需要注意：

- 每个人需要单独登录
- Token 是独立的，互不影响
- 建议设置不同的密码
- 敏感操作会记录日志，可追溯

### Q9: 如何查看操作日志？

**A**: 在云开发控制台中查看。

**查看方法：**
1. 进入云开发控制台
2. 点击「云函数」
3. 选择对应的云函数
4. 点击「日志」标签
5. 可以查看调用日志、错误信息等

### Q10: 管理后台访问慢怎么办？

**A**: 可能的原因和解决方法：

- **网络问题**：检查网络连接
- **云函数冷启动**：首次访问较慢是正常的，后续会变快
- **数据量过大**：考虑分页查询
- **CDN 缓存**：静态托管已自动使用 CDN 加速
