# 题库模板说明

这套模板按两个用途拆分：

## 1. 直接导入用

这些文件可以直接整理后导入当前后台：

- `00_空白通用导入模板.csv`
- `01_通用混合题型示例模板.csv`
- `02_客观题导入模板.csv`
- `03_主观题导入模板.csv`

适用场景：

- 你已经在后台创建好科目
- 你准备按单个科目逐次导入
- 你准备把题目导入到某个专题或试卷

注意：

- 当前后台是“选择科目后导入”，所以直接导入模板里不需要再写科目列
- 如果一个 CSV 里混入多个科目，当前后台不会自动拆分

## 2. 多科目整理用

- `04_多科目题库规划模板.csv`

适用场景：

- 你还没有正式题库，正在批量整理多个考试科目
- 你准备先做总表，再按科目拆分
- 你要同时规划 `科目 / 专题 / 试卷 / 题型`

建议流程：

1. 先用 `04_多科目题库规划模板.csv` 汇总所有题源
2. 按科目拆分成独立 CSV
3. 再复制到 `00_空白通用导入模板.csv` 或其他导入模板中
4. 后台按科目分别导入

## 题型支持

当前模板统一支持：

- 单选题
- 多选题
- 判断题
- 填空题
- 简答题
- 开放题
- 论述题
- 计算题

说明：

- `开放题`、`论述题` 会按简答题逻辑导入
- `判断题` 选项列可以留空
- `填空题` 多个答案用 `|` 分隔
- `简答题 / 开放题 / 论述题 / 计算题` 选项列可以留空
